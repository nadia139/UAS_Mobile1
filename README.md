Prakiraan Cuaca adalah aplikasi prakiraan cuaca iOS yang menggabungkan teknologi prediksi modern dengan kontribusi komunitas. Aplikasi ini dirancang untuk memberikan informasi cuaca, real-time, dan personal dengan antarmuka yang intuitif dan mudah digunakan.
# Visi
Menjadi aplikasi cuaca terdepan yang mengutamakan akurasi melalui kolaborasi teknologi dan komunitas.
# Misi
Memberikan informasi cuaca yang akurat dan terkini
Melibatkan pengguna dalam meningkatkan kualitas data cuaca
Menyajikan pengalaman pengguna yang sederhana dan menyenangkan
Membangun komunitas yang peduli terhadap kondisi cuaca

# Fitur Inti
Deteksi Lokasi Otomatis :	Mendeteksi posisi pengguna secara otomatis tanpa input manual
Prakiraan 3 Hari :	Prediksi cuaca lengkap untuk 3 hari ke depan dengan detail suhu
Statistik Cuaca :	Informasi detail: kelembaban, kecepatan angin, peluang hujan
Pelaporan Komunitas :	Sistem crowdsourcing untuk melaporkan kondisi cuaca aktual
Pembaruan Real-time :	Data cuaca diperbarui secara berkala

# Fitur Tambahan
Notifikasi Cuaca Ekstrem ⚠️


Pengguna Baru
1. [INSTAL] → [BUKA APLIKASI]
2. [PERMISI LOKASI] → "Izinkan akses lokasi?"
3. [LAYAR 1] → "Mendeteksi Lokasi Anda..."
4. [LAYAR 2] → "Halo! Cuaca cerah hari ini"
5. [LAYAR 3] → Konfirmasi: "Cerah, 28°C"
6. [LAYAR 4] → Dashboard utama dengan semua fitur

Kontribusi Komunitas
1. [DARI DASHBOARD] → Tap ikon "Masukan"
2. [LAYAR 5] → Pilih kondisi cuaca aktual
3. [SUBMIT] → Kirim laporan
4. [LAYAR 6] → Konfirmasi: "Terima kasih!"
5. [KEMBALI] → Ke dashboard dengan data diperbarui
